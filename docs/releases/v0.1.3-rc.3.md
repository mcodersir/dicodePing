# dicodePing v0.1.3-rc.3

پیش‌انتشار سوم نسخه 0.1.3 با تمرکز روی نصب پایدار Android و تجربه بهتر نسخه Windows.

## Windows

- اصلاح کامل دکمه تازه‌سازی پینگ با TCP probe روی پورت واقعی هر کانفیگ
- استفاده از میانه چند نمونه برای حذف پینگ‌های تصادفی و غیرواقعی
- اضافه‌شدن دکمه «پینگ سرور انتخاب‌شده»
- نمایش متن «اتصال» روی دکمه اصلی صفحه خانه
- بهبود ریسپانسیو جدول سرورها و جلوگیری از نصفه‌شدن دکمه اتصال
- حفظ منطق رد پینگ‌های غیرقابل‌اعتماد در انتخاب خودکار

## Android

- ساخت Release با JKS پایدار قبلی به‌جای کلید Debug
- افزایش versionCode برای نصب به‌صورت به‌روزرسانی روی نسخه موجود
- انتشار فقط یک APK یونیورسال برای Android 7 به بالا
- پشتیبانی از armeabi-v7a، arm64-v8a، x86 و x86_64

## فایل‌ها

- `dicodePing-v0.1.3-windows.exe`
- `dicodePing-v0.1.3-android.apk`
- `SHA256SUMS`

این نسخه Pre-release است و برای تست عمومی پیش از انتشار پایدار ارائه می‌شود.
