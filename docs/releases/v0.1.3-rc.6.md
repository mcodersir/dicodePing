# dicodePing v0.1.3-rc.6

Pre-release builds for Android, Windows, and Linux GUI.

## Highlights

- Android refresh now has three explicit stages: concurrent source download, location resolution, and bounded concurrent real Xray latency tests.
- Android server rows stay visible and stable while live latency results animate in place; the final ranking is applied once after all probes finish.
- Subscription downloads, DNS, Geo lookup, and real proxy tests use separate bounded concurrency limits, response timeouts, caching, and input size protection.
- Android `versionCode` advances to 9 while retaining the existing package id and signing certificate.
- Windows/Linux background discovery and Geo jobs now stop cooperatively and the GUI no longer destroys an active worker during shutdown.
- Linux ships the same PySide6 GUI and shared desktop connection logic as Windows.

## Android upgrade note

This APK is signed with the established dicodePing release certificate. If Android reports a conflicting package, remove the previously installed build first, then install this APK. Removing an app also removes its locally stored settings.
