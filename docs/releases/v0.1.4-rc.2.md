# dicodePing v0.1.4-rc.2

Pre-release for Windows, Linux and Android.

- Fixes server-subscription download failures caused by direct-only networking and temporary DNS failures.
- Uses the operating system proxy/PAC setting before direct access, attempts multiple official delivery paths, and safely reuses the last verified subscription when all network paths are unavailable.
- Keeps the existing server list intact while the source is temporarily unreachable.

Android installers are signed updates. If Android reports a package conflict, uninstall the older build first, then install this release.
