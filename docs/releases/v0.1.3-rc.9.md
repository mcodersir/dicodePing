# dicodePing v0.1.3-rc.9

این پیش‌انتشار یک اصلاح اضطراری برای شروع نسخه Windows است. مشکل ماندن دائمی روی پیام «در حال آماده‌سازی رابط کاربری» برطرف شده و مسیر شروع برنامه از عملیات شبکه جدا شده است.

## اصلاحات Windows

- دریافت سرورها، سنجش پاسخ و تشخیص موقعیت دیگر شروع رابط را مسدود نمی‌کنند.
- پنجره اصلی ابتدا با حالت سبک باز می‌شود و فهرست ذخیره‌شده بعد از نمایش بارگذاری می‌شود.
- یک زمان‌بان چهارثانیه‌ای در صورت کندی یا خرابی آماده‌سازی، رابط را با تنظیمات امن باز می‌کند.
- مسیرهای هم‌زمان پایان worker و زمان‌بان با یک دروازه idempotent کنترل می‌شوند تا دو پنجره ساخته نشود.
- خطای ساخت رابط مدیریت می‌شود و اسپلش در تمام مسیرهای موفق یا ناموفق بسته می‌شود.
- بارگذاری فهرست اولیه به ۳۲۰ ردیف محدود شده تا داده‌های بسیار بزرگ شروع برنامه را قفل نکنند.
- چرخه عمر worker هنگام خروج امن شده و از حذف thread در حال اجرا جلوگیری می‌شود.
- smoke test واقعی به فایل نهایی Windows اضافه شده است؛ CI باید پنجره اصلی را بسازد، نمایش دهد و با کد موفق خارج شود.

## Linux و Android

- Linux با اصلاحات مشترک دسکتاپ و همان smoke test واقعی رابط، دوباره ساخته شده است.
- منطق Android تغییر نکرده است؛ فقط `versionCode` به ۱۲ افزایش یافته و APK با گواهی رسمی پروژه دوباره امضا شده است.

اگر Android هنگام نصب تداخل بسته گزارش کرد، نسخه قبلی را حذف و سپس APK جدید را نصب کنید. حذف نسخه قبلی داده‌های محلی برنامه را نیز پاک می‌کند.
