# dicodePing v0.1.5-rc.2

این نسخه، مشکل بحرانی خالی‌ماندن فهرست سرورها بعد از Splash را در Windows و Linux برطرف می‌کند.

- سرورها بلافاصله پس از پردازش در فهرست قرار می‌گیرند؛ تست خودکار فایل نهایی نیز به انتشار اضافه شد.
- پینگ هر سه نسخه فقط پس از عبور ترافیک واقعی از همان کانفیگ معتبر می‌شود.
- کانفیگ ناموفق دیگر پینگ سالم نمی‌گیرد و وارد انتخاب خودکار نمی‌شود.
- بررسی اتصال Windows و Linux از مسیر خود برنامه انجام می‌شود و Proxy سیستم نتیجه کاذب ایجاد نمی‌کند.

دانلود: فایل `windows.exe` برای Windows، فایل `linux-x86_64.tar.gz` برای Linux و فایل `android.apk` برای Android است.

اگر نصب APK با تداخل بسته روبه‌رو شد، نسخه قبلی را حذف و سپس APK جدید را نصب کنید.
