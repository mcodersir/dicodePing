# dicodePing v0.1.3-rc.4

Pre-release for Windows reliability and Android package continuity.

- Android package is signed with the established application certificate and advances only `versionCode` to 7.
- Windows keeps subscription display names, shows the selected server on Home, and no longer changes Auto mode just by selecting a row.
- Automatic selection uses a successful endpoint latency probe and is no longer blocked by uncertain IP geolocation.
- Refresh deduplicates repeated endpoint probes, uses bounded parallel TCP tests, and resolves location metadata separately.
- TUN verification races multiple connectivity endpoints with a bounded startup budget to avoid false path errors from one slow endpoint.
