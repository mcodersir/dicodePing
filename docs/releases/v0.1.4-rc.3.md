# dicodePing v0.1.4-rc.3

Pre-release for Windows, Linux and Android.

- Checks for a matching platform update during splash startup and shows an action before entering the app.
- The update action opens the exact signed Windows, Linux, or Android release asset.
- Checks changed server subscriptions on splash startup and, after confirmation, downloads and tests the refreshed server list.
- Keeps startup responsive with a strict time limit when update services are unavailable.

Android installers are signed updates. If Android reports a package conflict, uninstall the older build first, then install this release.
