# dicodePing v0.1.5-rc.4

این نسخه یک اصلاح فوری برای مشکل «اتصال برقرار است اما سایت‌ها باز نمی‌شوند» است.

- مسیر DNS ویندوز، لینوکس و اندروید به حالت پایدار و امتحان‌شده بازگردانده شد.
- بررسی واقعی اتصال و بهبودهای سرعت پینگ نسخه قبل حفظ شده‌اند.
- هر سه خروجی ویندوز، لینوکس و اندروید دوباره ساخته و آزمایش شده‌اند.

اگر نصب APK به‌دلیل تداخل امضا انجام نشد، نسخه قبلی را حذف و سپس این نسخه را نصب کنید.
