# dicodePing v0.1.2

## فارسی

این نسخه همان شاخهٔ پایدار Windows و Android است و یک اصلاح اختصاصی برای نمایش صحیح آیکون در taskbar، Alt+Tab و فایل اجرایی ویندوز دارد.

### فایل‌ها

- `dicodePing-v0.1.2-windows.exe`: نسخهٔ ویندوز با دسترسی Administrator برای ساخت رابط TUN.
- `dicodePing-v0.1.2-android-debug-signed.apk`: APK قابل نصب برای آزمایش و استفاده مستقیم؛ با کلید debug امضا شده است.
- `dicodePing-v0.1.2-android-release-unsigned.apk`: خروجی release بدون کلید خصوصی؛ برای انتشار فروشگاهی باید با کلید مالک پروژه امضا شود.
- `SHA256SUMS`: هش SHA-256 تمام فایل‌های خروجی.
- `SBOM.spdx.json`: فهرست اجزای نرم‌افزار با قالب SPDX.
- `dicodePing-v0.1.2-licenses.zip`: متن مجوزها و اعلان‌های اجزای ثالث.
- `THIRD_PARTY_NOTICES.md`: منشأ، نسخه و مجوز coreهای Windows و Android.

### اصلاح آیکون ویندوز

- آیکون چنداندازه در resource فایل EXE ثبت می‌شود.
- آیکون سراسری `QApplication` پیش از ایجاد پنجره‌ها تنظیم می‌شود.
- `AppUserModelID` ثابت پیش از ساخت UI اعمال می‌شود.
- آیکون‌های native کوچک و بزرگ HWND پس از نمایش پنجره تنظیم می‌شوند.

## English

This release keeps the audited Windows and Android code line and adds a dedicated Windows shell-icon fix for the executable, taskbar, and Alt+Tab.

The release is built entirely by GitHub Actions. It publishes SHA-256 checksums, an SPDX SBOM, and GitHub artifact attestations. The Android debug APK is installable; the release APK intentionally remains unsigned because no private production signing key is stored in the repository.

Native dependencies are fetched from version-pinned upstream Releases during the build rather than committed as opaque binaries. Xray and Wintun integrity checks run before the Windows package is produced; the Android AAR is likewise pinned and verified before Gradle starts.
