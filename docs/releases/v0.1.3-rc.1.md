# dicodePing v0.1.3-rc.1 — پیش‌انتشار

این نسخه پیش‌انتشار برای آزمایش اصلاحات اتصال و بهبود تجربه کاربری Windows و Android ارائه شده است.

## اصلاحات مهم Windows

- حذف وابستگی اعتبارسنجی اتصال به حرکت فوری شمارنده‌های StatsService؛ این رفتار باعث ردشدن اشتباه اتصال‌های سالم در Windows 11 می‌شد.
- بررسی اتصال با چند درخواست واقعی بعد از آماده‌شدن مسیر TUN.
- افزایش زمان آماده‌سازی مسیر و پاک‌سازی DNS پیش از اعتبارسنجی.
- پیام خطای دقیق‌تر برای تفاوت بین آماده‌نشدن TUN و پاسخ‌ندادن سرور.
- حفظ آمار دانلود و آپلود به‌عنوان داده نمایشی، نه شرط موفقیت اتصال.

## بهبودهای Android

- ادامه استفاده از تست واقعی Xray و دریافت لوکیشن پیش از انتخاب خودکار سرور.
- یک APK یونیورسال برای Android 7 و بالاتر و چهار معماری اصلی.
- امضای پایدار JKS با بررسی اثرانگشت گواهی در CI.
- جابه‌جایی نرم‌تر بین صفحات بدون بازسازی غیرضروری Fragmentها.
- حفظ شبکه اصلی خارج از VPN برای جلوگیری از حلقه مسیریابی.

## فایل‌های نصب

- `dicodePing-v0.1.3-windows.exe`
- `dicodePing-v0.1.3-android.apk`

این نسخه Pre-release است و خروجی‌ها توسط pipeline اصلی CI برای Windows و Android بررسی می‌شوند. گزارش دستگاه، نسخه Windows/Android، نوع اتصال و فایل لاگ در رفع ایرادهای باقی‌مانده مفید است.

# dicodePing v0.1.3-rc.1 — Pre-release

This release candidate focuses on Windows tunnel validation reliability and general Windows/Android usability improvements.

## Highlights

- Stops treating delayed Xray traffic counters as a connection failure on Windows.
- Validates the active TUN route with repeated real network requests.
- Flushes stale Windows DNS state before validation and allows additional route warm-up time.
- Keeps traffic counters as telemetry rather than a connection gate.
- Preserves Android real-path server probing and automatic geolocation.
- Ships one universal, JKS-signed Android APK for Android 7+.
- Adds smoother Android page transitions while retaining fragment state.
- Uses the primary CI workflow for final Windows and Android verification before publishing.
