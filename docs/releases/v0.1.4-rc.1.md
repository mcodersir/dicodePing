# dicodePing v0.1.4-rc.1

Pre-release for Windows, Linux and Android.

- Adds an in-app update prompt that opens the matching signed release asset.
- Checks server-source revisions at startup and asks before downloading changed sources.
- Windows/Linux retain the server list while response times are refreshed; results update in place and are sorted only when complete.
- Android opens cached lists immediately, batches live ping updates to avoid RecyclerView lag, and keeps initial setup bounded to first launch.

Android installers are signed updates. If Android reports a package conflict, uninstall the older build first, then install this release.
