# dicodePing v0.1.3-rc.7

این نسخه یک پیش‌انتشار برای ویندوز، لینوکس و اندروید است.

- نام سرور دقیقاً از متن بعد از `#` خوانده می‌شود و نام‌های بلند بدون شکستن چیدمان کوتاه می‌شوند.
- دریافت، مکان‌یابی و سنجش پاسخ سرورها به‌صورت مرحله‌ای، دسته‌ای، هم‌زمان و با تلاش دوباره برای موارد ناموفق انجام می‌شود.
- انتخاب خودکار، گزینه‌های سالم و سریع را رتبه‌بندی می‌کند و در صورت شکست گزینه‌های جایگزین متفاوت را امتحان می‌کند.
- پنجره دسکتاپ از لبه‌ها با ماوس تغییر اندازه می‌دهد و فهرست سرورها و تنظیمات در عرض‌های کوچک واکنش‌گرا هستند.
- تنظیمات کارایی و گزارش تشخیصی سبک افزوده شده است. گزارش‌گیری به‌صورت پیش‌فرض خاموش است.
- متن‌های فنی غیرضروری از رابط کاربر حذف شده‌اند.

## نصب اندروید

APK با گواهی رسمی همین پروژه امضا شده است. اگر اندروید به‌دلیل تداخل بسته اجازه به‌روزرسانی نداد، نسخه قبلی را حذف و سپس این APK را نصب کنید. حذف نسخه قبلی، داده‌های ذخیره‌شده آن نسخه را نیز پاک می‌کند.
