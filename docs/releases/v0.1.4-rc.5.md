# dicodePing v0.1.4-rc.5

نسخه پیش انتشار برای ویندوز، لینوکس و اندروید.

## اصلاح مهم به‌روزرسانی

- مقایسه نسخه‌های RC اصلاح شد تا نسخه‌های جدید همان سری به‌درستی تشخیص داده شوند.
- یک مسیر سازگاری موقت برای RC.3 و نسخه‌های قدیمی‌تر اضافه شد؛ آن‌ها نیز اعلان به‌روزرسانی می‌گیرند و به فایل‌های همین RC.5 هدایت می‌شوند.
- نسخه‌های جدید این مسیر سازگاری را نادیده می‌گیرند و فقط انتشار واقعی پلتفرم خود را نمایش می‌دهند.

فایل اندروید امضاشده است. اگر اندروید خطای تداخل بسته نشان داد، نسخه قبلی را حذف و سپس این نسخه را نصب کنید.
